## HairRemoval
Implementation of hair removal in CUDA. The methods of hair detection refer to the paper[1][2].

### Requirements
* OpenCV 4.2.0
* CUDA 10.2

## Test Result  

Method            |Image Size     | Elapsed Time
------------------|:-------------:|--------------
Naive             |1.2 Megapixels |  18789 ms
GPU, OpenMp, AVX  |1.2 Megapixels |  325 ms
